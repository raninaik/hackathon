# Arcade Quick Setup for Impact Snapshot

## Desktop vs Chrome Extension: Which to Choose?

**Desktop App** 📱
- ✅ Smoother video + audio recording
- ✅ Better performance for longer recordings
- ✅ More stable screen capture
- ❌ Requires download and installation

**Chrome Extension** 🌐
- ✅ Fastest install (just add to browser)
- ✅ Quick setup for immediate use
- ✅ No app download needed
- ❌ Browser-dependent performance

**Recommendation**: Use Desktop for polished demos, Extension for quick proof-of-concepts.

## Installation Steps

### macOS
1. Go to [usearcade.com](https://usearcade.com)
2. Click "Download for Mac"
3. Open the downloaded `.dmg` file
4. Drag Arcade to Applications folder
5. Launch Arcade from Applications
6. Sign in with your Google account

### Windows
1. Go to [usearcade.com](https://usearcade.com)
2. Click "Download for Windows"
3. Run the downloaded `.exe` installer
4. Follow installation prompts
5. Launch Arcade from Start menu
6. Sign in with your Google account

### Chrome Extension (Alternative)
1. Visit [Chrome Web Store](https://chrome.google.com/webstore)
2. Search "Arcade by Arcade"
3. Click "Add to Chrome"
4. Pin the extension to your toolbar
5. Click the Arcade icon and sign in with Google

## Create Your Space

1. After signing in, click "Create Space"
2. Name it **"Impact Snapshot"**
3. Set visibility to "Public" (for easy sharing)
4. Click "Create"

## Recording Settings

### Video Settings
- **Resolution**: 1080p (1920x1080)
- **Frame rate**: 30 fps
- **Quality**: High

### Audio Settings
- **System audio**: OFF (for first pass)
- **Microphone**: OFF (for first pass)
- **Noise reduction**: ON (if available)

### Visual Settings
- **Cursor highlight**: ON ✨
- **Click animations**: ON
- **Zoom on click**: Optional

### Recording Strategy
- **Keep it short**: 4–6 steps maximum
- **One action per step**: Don't rush
- **Clear transitions**: Pause between major actions
- **Focus on impact**: Show the key result/outcome

## 30-Second Sanity Test

### Quick Test Recording
1. **Start recording** any webpage (try Google.com)
2. **Perform 3-4 simple actions**:
   - Navigate to a page
   - Click a button
   - Scroll down
   - Click another element
3. **Stop recording** (should be 15-30 seconds)
4. **Let Arcade auto-split** into steps
5. **Accept AI-generated captions** (or edit quickly)
6. **Publish** the replay
7. **Copy the share link**

### Test Checklist ✅
- [ ] Link opens correctly on mobile phone
- [ ] Captions are visible and accurate
- [ ] Audio is off (as intended)
- [ ] Shows 4–6 clear steps
- [ ] Video quality looks sharp
- [ ] Loading time is reasonable

## Troubleshooting

### Common Issues
- **Blurry video**: Check recording resolution settings
- **No cursor visible**: Enable cursor highlight in settings
- **Too many steps**: Record slower with clear pauses
- **Audio bleeding through**: Double-check audio settings are OFF

### Best Practices
- **Test on mobile first**: Most users will view on phones
- **Keep captions concise**: 6-8 words maximum
- **Show end result**: Always end with the impact/outcome
- **Use consistent timing**: 2-3 seconds per step
