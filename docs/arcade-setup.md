# Arcade Quick Setup for Impact Snapshot

A quick guide to get you recording professional demo videos with Arcade in under 5 minutes.

## Desktop vs Chrome Extension: Which to Choose?

**Choose Desktop App if:**
- You want the smoothest video quality and audio recording
- You're creating polished demos for presentations or marketing
- You have time for a 2-minute install

**Choose Chrome Extension if:**
- You need to start recording immediately (30-second install)
- You're doing quick walkthroughs or internal demos
- You primarily demo web applications

## Installation Steps

### macOS
1. **Desktop**: Download from [arcade.software](https://arcade.software) → Install the .dmg
2. **Extension**: Visit [Chrome Web Store](https://chrome.google.com/webstore) → Search "Arcade" → Add to Chrome
3. **Sign in**: Click the Arcade icon → "Sign in with Google" → Authorize

### Windows
1. **Desktop**: Download from [arcade.software](https://arcade.software) → Run the .exe installer
2. **Extension**: Visit [Chrome Web Store](https://chrome.google.com/webstore) → Search "Arcade" → Add to Chrome
3. **Sign in**: Click the Arcade icon → "Sign in with Google" → Authorize

### Create Your Space
1. After signing in → Click "Create Space"
2. Name it **"Impact Snapshot"**
3. Set visibility to "Public" (so anyone can view your demos)

## Recording Settings

### Essential Settings for First Pass
- **Resolution**: 1080p (crisp but not huge files)
- **System Audio**: OFF (cleaner for voice-overs later)
- **Microphone**: OFF (add narration in post if needed)
- **Cursor Highlight**: ON (helps viewers follow along)
- **Step Length**: Keep each step 4–6 actions max

### Pro Tips
- Start each recording with a clean browser tab
- Move slowly and deliberately
- Pause 1 second between major actions
- Keep total demo under 2 minutes for best engagement

## 30-Second Sanity Test

Before recording your real demo, do this quick test:

1. **Record**: Open any webpage (like Google.com) → Click record → Type something → Click a few links → Stop recording
2. **Review**: Let Arcade auto-split into steps (should be 4–6 steps)
3. **Caption**: Accept the AI-generated captions (edit if needed)
4. **Publish**: Click "Publish" → Copy the share link
5. **Test**: Open link on your phone → Verify it loads, captions show, audio is off

## Pre-Demo Checklist

Before recording your actual Impact Snapshot demo:

- [ ] Link opens instantly on mobile device
- [ ] Captions are visible and accurate
- [ ] Audio is off (no system sounds)
- [ ] Total steps are 4–6 (not overwhelming)
- [ ] Each step has a clear, actionable title
- [ ] Video quality looks crisp at 1080p
- [ ] Share link is copied and ready to use

## Troubleshooting

**Video looks blurry?** → Switch to Desktop app for better quality
**Too many steps?** → Combine similar actions or re-record with fewer clicks
**Captions wrong?** → Edit them manually before publishing
**Link won't open?** → Check that Space is set to "Public"
